geodecision.osmquery.methods.get_OSM_poly(bbox, key, value='all', timeout=40, endpoint='https://overpass-api.de/api/interpreter')[source]¶

Get OSM data with key/value pair and that is Polygon by making queries on Overpass

Returns

Return type

GeoJSON Polygons FeatureCollection

Parameters

  • bbox(tuple) (-) –

    • bounding box for the query

    • must be (SOUTH, WEST, NORTH, EAST)

    • must be EPSG 4326 (WGS84) projection

    • ex: (45.772, 4.864, 45.778, 4.875)

  • key(str) (-) –

    • key for the OSM query

  • value(str) (-) –

    • value for the OSM query

  • timeout(int) (-) –

    • time in seconds for the timeout

    • default: 40

  • endpoint(str) (-) –

    • endpoint for the queries

    • default: “https://overpass-api.de/api/interpreter”
